Show Thumbnail on nautilus

Dede Dindin Qudsy  
2017-10-28  about 57 words  1 min 

based on arch wiki, “There is an ongoing issue with totem (the video thumbnailer for Nautilus) on computers with Intel GPUs. It will crash if gstreamer-vaapi is installed". to resolve this, install ffmpegthumbnailer and gst-libav

1
2
3

$ sudo pacman -S ffmpegthumbnailer gst-libav
$ rm -r ~/.cache/thumbnails/fail
$ killall -q nautilus

thumbnail will be generated now.
